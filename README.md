# 📊 E-Commerce Risk & Return Dashboard

## 📌 Project Overview

The E-Commerce Risk & Return Dashboard is an interactive Power BI dashboard designed to analyze sales performance, product returns, fraud risks, and customer ratings. The dashboard helps identify business risks, monitor return trends, and understand customer purchasing behavior through interactive visualizations.

---

## 🎯 Business Objective

The objective of this project is to help businesses:

- Monitor sales performance.
- Identify high return product categories.
- Detect fraud patterns based on payment methods.
- Track monthly order trends.
- Improve customer satisfaction through data-driven insights.

# 🔍 Key Insights

### 📦 Sales Analysis
- Processed more than **12,000 customer orders**.
- Generated **₹13.40 Lakhs** in total sales.
- Most orders fall within the **₹0–₹100 order value range**.

### 🔄 Return Analysis
- Overall return rate is **9.27%**.
- **Fashion** products have the highest return rate.
- **Pet Supplies** show the lowest return rate.

### 🚨 Fraud Analysis
- Fraud rate is **3.73%**.
- **Credit Card** payments recorded the highest fraud cases.
- Fraud activity is lower for Bank Transfer and Gift Card payments.

### ⭐ Customer Experience
- Average customer rating is **4.41**, indicating generally positive customer satisfaction.

### 📅 Monthly Trends
- Order volume remains stable throughout the year.
- Return and fraud trends vary month-to-month, highlighting periods that require closer monitoring.

---

# 💡 Business Recommendations

- Improve quality checks for high-return product categories.
- Strengthen fraud detection for high-risk payment methods.
- Monitor seasonal increases in returns and fraud.
- Promote products with low return rates.
- Use customer feedback to improve products and reduce returns.
